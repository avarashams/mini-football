# Mini Football game

----
Simple game based on OpenGL libraries with C++. Player has to score as many goals as possible in a limited time.

## Demo GUI
![img.png](assets/img.png)
## Controls
- W - Move Forward
- S - Move Backward
- A - Move Left
- D - Move Right
